# Delartech

Delartech is a web application built with Next.js and Tailwind CSS. It includes various components and pages to provide a comprehensive digital experience.

## Installation

To install the project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/ErfanShokrollahzadeh/Delartech.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Delartech
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server, run the following command:

```bash
npm run dev
```

To build the project for production, run the following command:

```bash
npm run build
```

To start the production server, run the following command:

```bash
npm start
```

If you encounter issues with peer dependencies while installing `react-leaflet`, use the following command in your project root:

```bash

```
